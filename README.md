# Library Management System

Spring Boot + PostgreSQL backend for managing library operations.

## Features
- Book catalog
- Borrow and return system
- Fine management
- Payment tracking
- Hold queue
- Reporting dashboard

## Tech Stack
Spring Boot
PostgreSQL
Gradle
pgAdmin

## Database
Includes:
- 140 books
- 420 copies
- 600 borrow records
- fines and payments

## Setup
1. Create database librarydb
2. Run schema.sql
3. Run sample_data.sql
4. Run Spring Boot app
